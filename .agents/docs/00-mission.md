# 00 Mission

Goal: produce reproducible astrological artifacts using repository-native recipes and skill wrappers.

Primary quality targets:

1. Reproducibility (same inputs -> same outputs)
2. Provenance integrity (`canonical_source` valid)
3. Operational clarity (clear run summaries + TaskLogs)
4. Minimal manual post-processing

Non-goals:

- Free-form astrological computation outside project recipes
- Hidden local-only changes without logging
