## ФУНДАМЕНТ — семантический слой (база значений)

> **Статус: базовая версия готова → `docs/semantic-base.md`** (2026-06-14). Содержит роли,
> операторы композиции, некоммутативность, терсовые таблицы значений, каталог антипаттернов,
> типовые ритуалы и критерий приёмки (воспроизводимость чтения из дока). Дальше — наполнение
> и (по желанию) векторизация. Ниже — исходное обоснование архитектуры.

> Возможно, важнее всех таймингов: без него «портативный рецепт» тайно опирается на знания
> модели-читателя, и другой ИИ разойдётся в смыслах. Сейчас значения метода **не записаны**
> (есть только словарь 12 фаз + таблицы достоинств). Архитектура — три части:
>
> 1. **Топики / значения** — что значит дом, планета, знак, аспект, точка. Многозначно (веер),
>    не одной строкой. Планеты-как-деятели (Марс, Венера…) в проекте отсутствуют — добавить.
> 2. **«Обычно указывает →»** — библиотека делинеации комбинаций (планета в доме/знаке, аспект).
>    Формулировка **вероятностная** («обычно», «склонно»), с диапазоном смыслов.
> 3. **Антипаттерны значений** — каталог ловушек. Главные: детерминированный коллапс
>    («Марс в 7 = развод»), **одновалентность** (8-й = только смерть), предрешённая валентность
>    (страсть vs конфликт), чтение вторичного значения как первичного, импорт смысла из чужой
>    системы домов/уровня. Уже произведённые этим сеансом дисциплины (уровни, правило трёх,
>    запрет подгонки, «дешёвое» подтверждение, пустое Солнце-Солнце) — **собрать сюда в каталог**.
>
> Принцип записи: значение = **вероятностное и многозначное, валентность не предрешена**
> (прямое продолжение двухосной модели и правила «не сочинять вперёд, а наблюдать»).

### Некоммутативность — важнейшее правило (в фундамент)
Знаки/дома/планеты = **семантические резервуары** (векторы смысла). Делинеация = их композиция
(«перемножение матриц»), и она **некоммутативна**: порядок и направление меняют смысл (AB ≠ BA).

- **Семантическая (направление связи):** «2 связано с 8» ≠ «8 связано с 2». Доход от общего/чужого
  (2-й **от** 8-го) ≠ кризис/трата своего (8-й **от** 2-го) — **даже когда дериватив указывает в
  один дом** (оба → 9-й), смысловой путь разный. Объединяет уже встроенное направленное: аспект
  (быстрая→медленная), диспозиция (A правит знаком B, не наоборот), путь производных домов (N-й *от* дома).
- **Процедурная (порядок ритуала):** порядок чтения инструментов меняет результат.
  соляр→натал ≠ натал→соляр (что прочитано первым — задаёт рамку, второе читается внутри неё).
  Для ИИ порядок операций так же неравнозначен.

Следствие-артефакт: **типовые последовательности ритуалов** — канонические порядки чтения под
типовые задачи (напр. «год вперёд»: натал → профекция → соляр → прогрессия → транзит), с явной
пометкой, что задаёт рамку для чего. Композиция значений всегда **направленная**, не симметричная.

## Запланировано — структура натала
(порядок внутри свободный; «я в них не шарю» → краткое пояснение)

- **Неподвижные звёзды** — соединения натальных планет/углов с яркими звёздами (Алголь, Регул, Спика, Антарес, Сириус…) дают резкий «звёздный» акцент. У тебя уже всплыл **Уран △ Сириус 0.04°** — не прочитано системно.
- **Жребии** (кроме Фортуны) — вычисляемые точки: **Дух** (обратный Фортуне), Эрос, Необходимость и др. Тонкие смысловые маркеры; Дух нужен для Zodiacal Releasing.
- **Средние точки + антисции** — середина между двумя планетами (космобиология Эбертина) работает как чувствительная точка; **антисция** = зеркало через ось Рак–Козерог, скрытая связь планет. Рецепт упоминает антисцию как косвенный канал — не считали.
- **Малые достоинства + альмутен** — триплицитет/термы/лица: тонкие уровни «силы» планеты по градусу; **альмутен** = планета с наибольшим суммарным достоинством в точке (альмутен ASC/карты = более точный «управитель»). В рецепте осознанно отложены.
- **Целознаковые дома как наложение** — дом = весь знак, считая от знака ASC. Традиционные техники (профекции, ZR, секта) работают именно на нём, не на Placidus. Стоит наложить и сверить с Placidus.
- **Производные (повёрнутые) дома** — любой дом делается «первым», и от него считаются топики → у каждого дома появляются **вторичные значения**. Правило: 2-й **от** дома = его ресурсы, 4-й = основания/итог, 7-й = партнёр, 10-й = статус, 12-й = потери. Примеры: деньги партнёра = 2-й от 7-го = **8-й**; работа партнёра = 6-й от 7-го = **12-й**; ребёнок брата = 5-й от 3-го = **7-й**. Обычно крутят по целознаковым домам.

## Запланировано — тайминг
- **Затмения года** — солнечные/лунные затмения 2026–27, попадающие на натальные точки или оси соляра. Крупные «переключатели». Проверить, есть ли точные попадания.
- **Лунары** (месячные возвраты Луны) — карта на возврат Луны к натальной долготе (~раз в 27 дней). Таймер тоньше соляра, на месяц.
- **Фаза прогрессивного лунного цикла** — где ты в 30-летнем цикле прогр. Солнце–Луна (новолуние→первая четверть→полнолуние→…). Крупная фаза жизни: сев / рост / жатва / отпускание.
- **Zodiacal Releasing / Фирдария** — периоды-«главы» жизни: ZR (хеллинистика, от Жребиев Духа/Фортуны) и Фирдария (персидские планетные периоды). Кто «управляет» текущей главой жизни. Требуют Жребиев (см. выше) и/или секты.
- **Луна без курса (void-of-course / sine cursu)** — Луна не образует больше ни одного точного птолемеевского аспекта до выхода из знака. Родной дом — **электика/хорар** («дело не сдвинется»), НЕ ядро предсказательной натальной работы. Два РАЗНЫХ под-инструмента, не путать:
  - **Транзитная Луна без курса** — суточный электикальный маркер (несколько часов раз в пару дней): качество момента, «запущенное в холостую Луну не пристаёт». Разрешение — **часы**, тоньше годового скана; считать только если нужен «хороший момент действовать». Дёшево: аспекты Луны к классическим планетам в текущем знаке + проверка, остаётся ли применяющий до границы знака.
  - **Прогрессивная Луна без курса** — прогр. Луна (≈1°/мес) перед сменой знака без новых прогрессивных аспектов = «межглавье», затишье. **Модерновое наложение**, мягкое, масштаб — месяцы; не строгая классика.
  - **Пометка-антипаттерн:** в дирекциях **градус-год** (solar arc / примарные) «без курса» НЕ используется — дирекции работают перфекцией дуги, а не «исчерпанием аспектов в знаке». Не переносить туда.
  - **Уровень:** быстрый (транзитная) или мягкий месячный (прогрессивная) маркер; в стоячую структуру и в годовую рамку не входит — вопрос разрешения, не пропущенная база.

## Примечание по уровням
- **Структура натала** (звёзды, жребии, средние точки, малые достоинства, секта, целознак) — читается один раз, описывает стоячую форму.
- **Тайминг** (solar arc, профекции, затмения, лунары, прогр. лунная фаза, ZR/Фирдария, луна без курса) — отвечает «когда/какой период», как транзит/прогрессия/соляр.
- Дисциплина прежняя: стоячее трактуем для понимания, окна вперёд — наблюдаем и сверяем, не подгоняем.
