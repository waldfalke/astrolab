# ASTRO_GLOSSARY (формальный, многоязычный)

Назначение: рабочий глоссарий для вычислительного контура (карта, дома, аспекты, прогностика), без бытовых трактовок.

Правило: если в источниках есть расхождения школ, для проекта берется нейтральная вычислительная формулировка.

| G23 | Транзит | Astrological transit | de: Transit (Astrologie), fr: transit astrologique | Текущее движение планет относительно натальной карты (в т.ч. по аспектам к натальным точкам). |

| G33 | Сидерический зодиак | Sidereal zodiac | fr: zodiaque sidéral, de: siderischer Tierkreis | Зодиак, привязанный к звездному фону; учитывает прецессионный сдвиг через ayanamsa. |
| G34 | Прецессия равноденствий | Precession of the equinoxes | fr: précession des équinoxes, de: Präzession | Медленный дрейф оси Земли/точек равноденствий, из-за которого тропическая и сидерическая системы расходятся. |

1. Для расчета используем только термины G01..G34.
2. Любой новый термин сначала добавляется в этот файл (RU/EN + минимум один иноязычный алиас).
3. Если по термину есть расхождение школ, фиксируем выбор в отдельном `docs/ASTRO_SPEC_NOTES.md`.
