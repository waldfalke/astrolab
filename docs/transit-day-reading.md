# Чтение транзитов «на день»

Как читать транзиты на конкретный день (не на год). Канонический разбор — NKS realm `astrolab` #90;
этот файл — рабочий конспект. Смежное: `report-standards.md`, `semantic-base.md`, углы — NKS #88.

## Главное: день — это ДИАПАЗОН СУТОК, не снимок на полдень

Снимок на полдень — это момент. У него две беды: **углы и транзитные дома** за сутки прокручиваются
через весь зодиак (ASC ~1°/4 мин → 360° за день) — «дом дня» как одно число не существует; а **планеты**
за день движутся по-разному. Поэтому день читается как **события суток**, а не одна карта.

## Три стрелки времени (по скорости планеты)

| стрелка | кто | роль в дне |
|---|---|---|
| **Фон** | медленные (♄♅♆♇ + дирекции) | за сутки почти не двигаются → в орбе весь день, ровный фон (часто — годовой перевал; день лишь подсвечивает) |
| **Часовая** | **Луна** (~13°/сут) | единственный настоящий «оконщик» внутри дня: её аспект в орбе ~3–4 ч → события суток с часами (аспекты к наталу, смена знака, VOC) |
| **Тонкая** | **восходящий** (ASC/MC, ~1°/4 мин) | транзитный ASC/MC проходят натал-точки → активации до минут: ASC = тема **восходит**, MC = **кульминирует** |

**«Твой момент дня»:** когда транзитный ASC = натал-ASC и транзитный MC = натал-MC (≈один момент за
сутки) — небо повторяет ориентацию карты рождения для этого места.

## Окна и масштаб — ключевая мысль

«Оконщик» — тот, чья скорость соизмерима со шкалой:
- **День** → окна делает **только Луна** (часовые). Быстрые ☉☿♀♂ за сутки ~1° → их аспект-окно длится
  **2–6 дней** (шире суток): внутри одного дня они почти-фон, перфекция — на шкале **дней**, не часов.
  Медленные — фон ещё шире (недели).
- **Год** → окна делают **медленные** (их аспект тянется недели-месяцы), а внутри дня они застывают в фон.

То есть роли «окно/фон» **переворачиваются** с масштабом. Не путать день с годом.

## Углы — первоклассны (NKS #88)

Транзит к натальному **углу** (ASC-DSC, MC-IC) весомее того же к рядовой планете. **Транзитные дома**
(куда сегодня падают планеты) крутятся за сутки — как слой дня их нет; **натальные углы** неподвижны —
транзит к ним валиден. При точном времени — твёрдое ●; при приблизительном — ◐ (надёжность ≠ вес).

## Сферы в дне — только краткий доменный срез

Не послойный разбор каждой сферы (день мал — вышла бы вата/Barnum), а короткая сводка «куда сегодня
погода»: громко / средне / тихо по 7 узнаваемым областям. Часто прямо следует из того, какие **углы и
точки** задеты.

## Подача и реализация

Порядок подачи: **методология «день»** → **фон** → **тон дня** (быстрые касания, перфекции рядом) →
**Луна** (часовая стрелка) → **углы** → **восходящий день** (тонкая стрелка, опц.) → **срез по областям**
→ **итог**. Сборка пока разовая (swiss по часам для ASC/MC, transit-to-natal в орбе, HTML→PDF); рецепта
`transit_day` нет — кандидат, если запросы повторятся.
