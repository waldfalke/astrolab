# Task Log — L402-MVP-008: Implement LND with Healthchecks

**Date:** 2026-03-06
**Status:** DONE

## Objective

Развернуть стек `bitcoind` + `lnd` с использованием `healthcheck` в Docker Compose для стабильного запуска без гонки состояний.

## Completed

1. Добавлен `-rpcbind=0.0.0.0` в bitcoind — без него RPC слушал только 127.0.0.1, и LND из соседнего контейнера получал `connection refused`.
2. Добавлен `--tlsextradomain=lnd` в LND — без него TLS-сертификат не содержал hostname `lnd`, mock-api не мог установить gRPC-соединение (`ERR_TLS_CERT_ALTNAME_INVALID`).
3. Удалён устаревший `version: '3.7'` из docker-compose.yml.
4. Healthcheck-цепочка `bitcoind (healthy) -> lnd (healthy) -> mock-api` работает корректно.

## Root Causes (3 блокера)

| # | Симптом | Причина | Фикс |
|---|---|---|---|
| 1 | LND: `dial tcp bitcoind:18443: connection refused` | bitcoind слушал RPC на 127.0.0.1, не на 0.0.0.0 | `-rpcbind=0.0.0.0` |
| 2 | mock-api: `ENOENT rpc.proto` | Неверное имя proto-файла в server.js | `rpc.proto` -> `lightning.proto` |
| 3 | mock-api: `ERR_TLS_CERT_ALTNAME_INVALID` | TLS-сертификат LND не включал DNS:lnd | `--tlsextradomain=lnd` + пересоздание volume |

## Self-Check

### What was not done?

1. Автоматическое создание bitcoind wallet при первом запуске (требуется ручной `createwallet` + `generatetoaddress 101`).

### What was weakly done?

1. Требуется ручная генерация блоков (`generatetoaddress 101`) для синхронизации LND. Можно автоматизировать через init-скрипт.

### How not to do it

1. Не использовать кастомные `entrypoint.sh` для ожидания сервисов — `healthcheck` + `depends_on: condition: service_healthy` работает надёжнее.
2. Не забывать про `rpcbind` при использовании bitcoind в multi-container сетапе.
3. При изменении TLS-параметров LND обязательно удалять старый volume (`lnd_data`), иначе старый сертификат не пересоздаётся.

### How to do it properly

1. Всегда добавлять `--tlsextradomain=<service_name>` при запуске LND в Docker Compose.
2. Всегда добавлять `-rpcbind=0.0.0.0` для bitcoind в multi-container среде.
