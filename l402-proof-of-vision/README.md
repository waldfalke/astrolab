# L402 Proof of Vision

Этот проект является техническим "дымовым тестом" для проверки гипотезы aA2A (Automated Agent-to-Agent) коммерции с использованием протокола L402.

**Цель:** Продемонстрировать, что программный агент может самостоятельно, без участия человека, оплатить API-вызов с помощью микротранзакции в криптовалюте.

## Запуск эксперимента

Для запуска теста убедитесь, что у вас установлен Docker и Docker Compose.

Выполните команду из корневой директории этого проекта (`l402-proof-of-vision`):
```bash
docker compose run --build client-agent
```
Эта команда соберет (при необходимости) образы для `mock-api` и `client-agent`, запустит их в одной сети и выполнит скрипт агента.

## Ожидаемый результат

В случае успеха, в вашей консоли должен появиться следующий лог, демонстрирующий, как агент сначала получает отказ в доступе (402), а затем успешно "оплачивает" его и получает платный ресурс:

```
[AGENT] 1. Пытаюсь получить доступ к платному ресурсу: http://mock-api:3000/sun_sign
[AGENT] 2. Получен ответ 402 Payment Required. Это ожидаемо.
[AGENT] 3. Получен заголовок с "счетом": L402 token="dummy_token_identifier" invoice="dummy_invoice_details"
[AGENT] 4. Извлечен токен: dummy_token_identifier. "Оплачиваю" счет, используя секретный preimage.
[AGENT] 5. Формирую заголовок авторизации: L402 dummy_token_identifier:dummy_preimage_secret_value
[AGENT] 6. Делаю повторный запрос с оплатой...
[AGENT] 7. УСПЕХ! Ресурс получен.
-------------------------------------------
          ДОКАЗАТЕЛЬСТВО ВИДЕНИЯ:
{
  "sign": "Leo",
  "date": "2026-03-06T14:19:37.408Z",
  "paid_resource": true
}
-------------------------------------------
```
